Название: Поле Password не обязательно для заполнения при регистрации
Предусловия: 
	1. Сайт поднят в среде по адресу http://localhost:5000
	2. Открыта страница регистрации http://localhost:5000/signup

Шаги:
	1. В поле Email ввести валидный Email не зарегистрированный ранее
	2. Поле Password оставить пустым
	3. Нажать на кнопку Sign Up

Фактический результат:
	1. Пользователь перенаправлен на страницу входа.
	2. Не отобразилось предупреждение об обязательности заполнения поля Password
	3. Пользователь зарегистрирован с пустым паролем

Ожидаемый результат:
	1. Отобразилось предупреждение об обязательности заполнения поля Password
	2. Пользователь с пустым паролем не зарегистрирован
	3. Текущая страница не изменилась (нет перенаправления на страницу /login)

Серьёзность дефекта: средняя

Приоритет исправления: высокий

Требование: R-03
Тест кейс: Т-SignUp-03
